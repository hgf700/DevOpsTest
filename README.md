# DevOpsTest

This repository is a DevOps practice project focused on containerization, automation, and infrastructure management.

## Project Overview

The main goal of this project was to gain hands-on experience with DevOps tools and workflows by working with small React applications and deploying them using modern DevOps practices.

## What Was Implemented

- Dockerization of small React applications  
- Creation of Kubernetes YAML manifests:
  - manually
  - automatically  
- CI/CD pipeline implementation using **GitHub Actions**  
- Application deployment using **Kubernetes**  
- Early-stage YAML files created to serve as **Infrastructure as Code (IaC)**  

## Technologies Used

- Docker  
- Kubernetes  
- GitHub Actions  
- React  
- YAML  

## Purpose

This project was created for learning and experimentation with DevOps concepts such as:
- containerization
- continuous integration and deployment
- Kubernetes orchestration
- infrastructure as code
